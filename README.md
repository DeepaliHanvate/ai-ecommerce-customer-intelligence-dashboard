# 🛒 AI-Powered E-Commerce Customer Intelligence Dashboard

🚀 End-to-End Data Analytics Project using Python & Power BI

---

## 📌 Project Overview
This project analyzes e-commerce customer data to generate actionable insights and improve business decision-making.

## 🎯 Objective
- Analyze customer purchasing behavior
- Identify high-value customers
- Improve revenue using data-driven insights

## 🛠️ Tools Used
- Power BI
- Python (Pandas, NumPy)
- Excel

---

## 🧹 Data Cleaning (Python)
- Removed null values
- Handled duplicates
- Data transformation using Python
- Created features like Average Order Value (AOV)
